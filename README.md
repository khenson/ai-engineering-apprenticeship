# AI Engineering Apprenticeship

60-day apprenticeship building AI-powered tools for small and medium-sized businesses.

## Projects

### Day 1 — Business Name Generator
Generates brandable business names using the Anthropic API.

### Day 2 — Lead Processor
Reads a CSV of leads and generates personalized outreach messages using Claude.

## Setup
1. Clone this repo
2. Create a virtual environment: `python3 -m venv venv`
3. Activate it: `source venv/bin/activate`
4. Install dependencies: `pip install anthropic python-dotenv`
5. Add your API key to a `.env` file: `ANTHROPIC_API_KEY=your-key-here`
